🗓️ **Weather&Planner**


A simple calendar app built with React, HTML and CSS that allows users to add events and view current weather information using a public weather API.


**Features**

Add, edit, and delete calendar events

Fetch and display current weather for selected location

Daily and weekly calendar view

Integration with external weather API



**Technologies Used**

React

HTML5 & CSS3

JavaScript (ES6+)

AccuWeather API

**Installation & Setup**

To run this project locally:

git clone https://github.com/oliwiaprzekurat/Daily.git

cd weather-calendar

npm install

npm start

The app will run in development mode at http://localhost:3000.
